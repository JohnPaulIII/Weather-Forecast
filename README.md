# Weather-Forecast
A ruby app that displays a 7 day weather forecast based on your location.

# Installation
### Clone the repository
```
git clone https://github.com/JohnPaulIII/Weather-Forecast.git
```
### Start the Server
```
cd Weather-Forecast
rails s
```
### View the Page
By default, the page is located at this address:
```
localhost:3000
```
